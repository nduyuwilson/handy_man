# Thitima - Electrical Component & Quotation Management

**Thitima** is a professional Android application designed for electrical contractors and site managers to manage electricity components, set up projects, track pricing, and generate professional PDF quotations for clients.

## 🚀 Features

### 📋 Project Management
- **Site Setup**: Set up projects with specific locations, descriptions, and client details.
- **Dynamic Materials**: Add items from your catalogue to projects with custom quantities.
- **Smart Logic**: Automatically detects duplicate items and offers to merge quantities to keep quotations clean.
- **Flexible Labour**: Set labour costs as a fixed amount or as a percentage (e.g., 25%) of the total material cost.
- **Rules of Engagement**: Include specific terms and conditions for each project.

### 📦 Catalogue & Inventory
- **Multi-Brand Support**: Add various brands (e.g., Schneider, Legrand) for a single component type (like sockets or bulb holders).
- **Visual Browsing**: Modern 2-column grid layout with high-quality images for easy identification.
- **Price Tracking**: Maintain both buying and selling prices for all variants.
- **Integrated Search**: Quickly find components or projects using the real-time search functionality.

### 📄 Professional PDF Quotations
- **Branded Invoices**: Generates industry-standard PDFs with themed headers and footers.
- **Custom Identity**: Fully editable Business/Company name that reflects throughout the app and PDF.
- **Security**: Professional multi-line watermarking (Business Name + Phone Number) to prevent unauthorized use of your documents.
- **One-Click Sharing**: Instantly send quotations via WhatsApp, Email, or Text as professional PDF attachments.

### 🛡️ Backup & Restore
- **Full ZIP Backup**: Export your entire database, including all project data, brand variants, and **internal images**, into a single ZIP file.
- **Profile Persistence**: Backs up your professional profile and payment settings (Bank, Paybill, Till).
- **Cross-Device Portability**: Restore your full environment on any new device seamlessly.
- **CSV Export**: Export your catalogue to CSV for management in Excel or Google Sheets.

### 🎨 Modern UI/UX
- **Material 3 Design**: Clean, modern interface following the latest industry standards.
- **Day/Night Mode**: Full support for light and dark themes based on your preference.
- **System Adaptive**: Designed to respect device notches and status bars for a balanced look on all phones.

---

## 🛠️ Tech Stack
- **Language**: Java
- **Architecture**: MVVM (Model-View-ViewModel)
- **Database**: Room Persistence Library
- **Networking/JSON**: GSON
- **Image Loading**: Glide
- **UI Components**: Material Components for Android (Material 3)
- **Navigation**: Jetpack Navigation Component
- **Reporting**: Android PDF Document API

## ⚙️ Installation & Setup
1. Clone the repository.
2. Open in Android Studio (Iguana or later recommended).
3. Ensure `compileSdk` is set to 36.
4. Sync Gradle and Run.
